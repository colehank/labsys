# 快速上手

欢迎使用 CIBOL 实验室系统。这里用几分钟带你过一遍主要功能，之后你就能自己上手了。

## 登录

用管理员发给你的**账号邮箱 + 初始密码**登录。首次登录后，建议到 **我的 → 安全** 修改密码。

> 忘记密码或登录不进去，直接找管理员重置，不要反复尝试。

## 侧边栏导航

左侧是主导航，收起时只显示图标、展开后带文字。常用入口：

- **首页** —— 你的概览：近期组会、待办、通知。
- **组会** —— 组会日程、汇报顺序、你负责的场次。
- **服务器** —— 网页终端，一键登录实验室服务器。
- **密钥管理** —— 申请与查看你的 API 密钥、用量。
- **文档** —— 就是你正在看的这里。

底部头像点开有 **消息**、**设置**、**退出登录**；管理员还能在底部切换「成员视图 / 管理员视图」。

## 组会

在 **组会** 页可以看到整学期的日程和汇报顺序。轮到你汇报时，会在首页和消息里提醒。如果你和别人需要**对调场次**，可以发起对调申请，对方确认后即生效。

## 报告评分

每次汇报结束后，请到问卷星给当次汇报人打分（态度 / 精良 / 逻辑清晰，各 1–5 分）。这些分数是[月度评优](#)的三个数据来源之一，认真打分能让评优更公平。

## 申请 API 密钥

到 **密钥管理 → 申请密钥**，填写备注名、用途和预算，提交后等管理员审批。通过后密钥会出现在本页，可直接复制使用，并能看到已用额度与剩余预算。

## 服务器终端

**服务器** 页是一个网页版终端。在「我的 → 安全 → 服务器账密」保存一次账号密码后，即可一键登录，不必每次手输。切换到别的页面时终端会话不会断开。

## 消息与通知

头像菜单里的 **消息** 汇总了站内通知、待你确认的对调请求等。红点表示有未读。

## 遇到问题

- 功能相关：先查本 **文档** 中心。
- 账号 / 权限 / 数据异常：找管理员。
- 想提意见又不想具名：用 **匿名意见**（管理员视图可见汇总）。
