# 年度日程与事项

实验室一年的固定节律，方便大家提前安排。具体日期以每学期公布的组会日程为准。

## 学期结构

- **春季学期** —— CIBOL 大组会按周进行，穿插进展汇报与文献精读。
- **秋季学期** —— 同样按周进行，延续汇报顺序轮转。

每学期开学时会发布完整的**汇报顺序表**，可在 **组会** 页查看自己负责的场次。

## 月度评优

- 每个自然月为一个**评选期**。
- 评选依据见[评分规则](#)：报告评分、出勤、讨论三来源，五维等权。
- 评选期结束后公布当月之星，并在首页与消息提醒。

## 组会类型

| 类型 | 是否评分 | 是否记考勤 |
| --- | --- | --- |
| 进展汇报 | 是 | 是 |
| 文献精读 | 是 | 是 |
| 工作坊 | 否 | 是 |
| 团建活动 | 否 | 是 |

不评分的活动（工作坊、团建）同样计入出勤，只是不产生报告评分。

## 其他固定事项

- **学期初** —— 发布汇报顺序、更新成员名单。
- **每月末** —— 结算月度评优。
- **学期末** —— 汇总一学期的组会与评优记录。

> 本页为框架性说明。带具体日期的安排，请以 **组会** 页当学期的日程为准。
